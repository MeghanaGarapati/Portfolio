## 🛠️ Portfolio Tech Stack

The architecture of this digital portfolio is built entirely using vanilla and highly performant frontend technologies, eliminating bloated dependencies:
- **Structure:** `HTML5` (Semantic elements for optimized SEO structure)
- **Styling & Layout:** `CSS3` (Custom variables, Keyframe animations, Glassmorphic cards) + `Bootstrap 5` (Flexible grid frameworks)
- **Interactivity:** `Vanilla JavaScript` (DOM manipulations, theme management, 3D tilt effects, intersection animations)
- **Icons & Visuals:** `FontAwesome v6` (High-definition vector icons) & `Devicons` (Technology-specific branding)

## 🎨 Design Features & Architecture

### 1. Dynamic Dark & Light Theme System
The interface integrates a custom CSS custom-property matrix enabling real-time theme shifts without layout shifts or reloading. 
*   **Default Mode:** Sleek, high-contrast dark space theme featuring muted neon accents.
*   **Light Mode:** Clean, vibrant, minimalist corporate aesthetic with soft shadow depths.
*   **Implementation:** Triggered via a custom interactive toggle button that dynamically updates the data attributes on the `<body>` element.

### 2. Immersive 3D Hover & Depth Effects
*   Leverages JavaScript-driven or hardware-accelerated pure CSS `transform: perspective() rotateX() rotateY()` configurations to create realistic depth adjustments when interacting with project and experience cards.
*   Soft, layer-based box shadows translate dynamically based on hover positions, delivering a polished, modern feel.

### 3. Layout Breakdown
*   **Hero / Introduction:** Bold personal branding statement accompanied by fluidly typed animation strings and visual tech badges.
*   **About Me & Tech Stack:** A balanced section communicating core methodologies alongside a comprehensive skill mapping structure.
*   **Academic Journey:** Interactive historical timelines presenting detailed metrics such as continuous Grade Point Averages.
*   **Experience & Internships:** Highly styled multi-layered professional entries detailing exact tooling ecosystems.
*   **Interactive Projects Showcase:** Comprehensive, image-ready layout windows supporting technical highlights and seamless media frames.
*   **Certifications Grid:** Dynamic grid layouts featuring stylized modular blocks highlighting multi-platform cloud and software credentials.
*   **Contact & Footers:** Fully integrated communication channels fitted with scalable vector interaction items.

## 🚀 Repository & Setup Instructions

To host or locally clone this portfolio repository onto your workstation, follow the instructions provided below:

### Prerequisites
Ensure a modern web browser is installed (e.g., Google Chrome, Mozilla Firefox, Microsoft Edge, Safari). No heavy runtime packages or installations are required.

### Local Development Setup
1. Clone the project locally:
   ```bash
   git clone https://github.com/your-username/meghana-portfolio.git
   ```
2. Navigate into the project folder:
   ```bash
   cd meghana-portfolio
   ```
3. Open the entry point file inside your browser or use a live server extension:
   ```bash
   # If using VS Code Live Server, right-click index.html and select 'Open with Live Server'
   # Or launch directly via terminal command line:
   open index.html
   ```
